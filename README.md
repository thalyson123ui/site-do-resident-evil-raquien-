# Resident Evil: Requiem - Site Oficial

Site conceito para o jogo Resident Evil: Requiem, desenvolvido como projeto de demonstração.

## 🚀 Funcionalidades

- **Design Responsivo**: Layout adaptável para desktop, tablet e mobile
- **Modo Escuro/Claro**: Toggle para alternar entre temas
- **Galeria de Imagens**: Modal interativo para visualizar lugares do jogo
- **Sinopses de Personagens**: Cards clicáveis com informações detalhadas
- **Música de Fundo**: Player de áudio com controle de volume
- **Navegação Suave**: Scroll behavior smooth e menu mobile

## 🎯 Melhorias Implementadas

### Acessibilidade
- ✅ Alt texts descritivos em todas as imagens
- ✅ Skip link para navegação rápida
- ✅ Foco visível em elementos interativos
- ✅ Suporte a navegação por teclado (Tab, setas, Escape)
- ✅ ARIA labels e roles apropriados
- ✅ Estrutura semântica HTML5
- ✅ Suporte a prefers-reduced-motion

### SEO
- ✅ Meta description e keywords otimizadas
- ✅ Open Graph e Twitter Cards
- ✅ Canonical URL
- ✅ Sitemap.xml e robots.txt
- ✅ Títulos e headings estruturados
- ✅ Schema markup básico

### Melhores Práticas
- ✅ Performance: Lazy loading, preload de recursos críticos
- ✅ Segurança: rel="noopener noreferrer" em links externos
- ✅ Progressive Enhancement
- ✅ Código limpo e organizado
- ✅ Compatibilidade cross-browser

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Grid, Flexbox, variáveis CSS, animações
- **JavaScript ES6+**: Interatividade e funcionalidades
- **Google Fonts**: Tipografia personalizada

## 📱 Como Usar

1. Clone o repositório
2. Abra `index.html` em um navegador moderno
3. Navegue pelas seções usando o menu ou scroll

## 🎮 Controles

- **Menu Mobile**: Botão hambúrguer no header
- **Galeria**: Clique nas imagens dos lugares
- **Modais**: Use setas do teclado ou botões para navegar
- **Música**: Botão de alto-falante para toggle
- **Tema**: Botão lua/sol para modo escuro/claro

## 📄 Estrutura do Projeto

```
/
├── index.html          # Página principal
├── style.css           # Estilos CSS
```
